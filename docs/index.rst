scikit-batch-rl - A data-driven deep reinforcement learning library as a tool.
==============================================================================

scikit-batch-rl is a easy-to-use data-driven deep reinforcement learning
library.

* scikit-batch-rl is desined for practical projects.
* scikit-batch-rl provides state-of-the-art techniques across RL researches to
  let everyone easily achieve an incredible performance.
* scikit-batch-rl provides scikit-learn-compatible APIs.
